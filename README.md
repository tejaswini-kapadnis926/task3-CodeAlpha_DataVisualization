# 📊 CodeAlpha Data Visualization Project

## 🚀 Project Overview
This project focuses on transforming raw e-commerce product data into meaningful visualizations and interactive dashboards using Python and Power BI.
The project helps analyze customer sentiment, product ratings, pricing trends, and brand performance through charts, graphs, and business dashboards.
---

# 🛠️ Tools & Technologies Used
* Python
* Pandas
* Matplotlib
* Seaborn
* Power BI
* TextBlob
* Jupyter Notebook
---

# 📈 Visualizations Created
* Product Rating Distribution
* Price Distribution Histogram
* Top Brands Analysis
* Sentiment Analysis Countplot
* Scatter Plot Analysis
* Interactive Power BI Dashboard
---

# 📊 Dashboard Features
✅ KPI Cards
✅ Product Rating Analysis
✅ Top 10 Brands
✅ Sentiment Analysis
✅ Interactive Filters & Slicers
✅ Business Insights Visualization
---

# 🎯 Key Insights
* Most products have ratings between 3 and 5.
* Positive customer sentiment dominates the dataset.
* Certain brands have significantly more product listings.
* Product prices mostly fall in low-to-medium ranges.
